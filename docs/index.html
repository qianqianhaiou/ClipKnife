常见问题

关于隐私、格式与使用方式的快速解答。

素刀 需要联网吗？

核心功能完全离线运行。搜索、索引、视频分镜与抽帧都在本地完成，素材文件不会上传至任何云端服务器。

支持哪些图片和视频格式？

图片支持 JPG、JPEG、PNG、WebP、BMP；视频支持 MP4、MOV、MKV、AVI、WebM。后续版本会持续扩展更多格式。

视频搜索是怎么实现的？

素刀 使用 TransNetV2 对视频进行分镜，再用 ffmpeg 抽取每个分镜的代表帧，最后将代表帧送入本地语义索引。搜索时，你看到的是代表帧，点击即可打开原视频并跳转到对应时间段。

可以添加整个盘符或外接硬盘吗？

可以。素刀 支持文件夹、整盘符以及外接硬盘路径作为素材源，并会针对盘符级扫描提供默认排除规则，避免扫描系统目录。

我的素材隐私如何保障？

所有索引文件、向量与缓存都保存在你的本地应用配置目录中；ClipKnife 不收集素材内容、不上传缩略图、不记录搜索历史到云端。
